## 🔔 Звуковые уведомления

### Сценарий использования:

```
Браузер 1 (Иван):                 Браузер 2 (Мария):
┌─────────────────────┐          ┌─────────────────────┐
│  💬 Общий чат       │          │  💬 Общий чат       │
├─────────────────────┤          ├─────────────────────┤
│                     │          │                     │
│  Привет! 👋         │          │  Привет! 👋         │
│                     │          │                     │
│                     │  ──────> │  🔔 *звук*          │
│                     │          │  Как дела?          │
│                     │          │                     │
└─────────────────────┘          └─────────────────────┘
```

**Что происходит:**
1. Мария отправляет сообщение
2. Иван слышит звук уведомления 🔔
3. Сообщение появляется в чате

## 🎯 Сравнение "До" и "После"

### Онлайн-статус:

| До | После |
|----|-------|
| Нет информации о статусе | 🟢 Онлайн индикатор |
| Не видно когда был онлайн | ⏰ "был 5 мин. назад" |
| Нет фильтрации | 🔍 Фильтр "Только онлайн" |

### Drag & Drop:

| До | После |
|----|-------|
| Только кнопка "Выбрать файл" | 📎 Перетаскивание файлов |
| Нет визуальной обратной связи | 🎨 Синяя зона с анимацией |
| Несколько кликов | ⚡ Один жест |

### Звуковые уведомления:

| До | После |
|----|-------|
| Только визуальные уведомления | 🔔 Звук + визуал |
| Легко пропустить сообщение | 🎵 Слышно даже в фоне |
| Нет настроек | 🔇 Можно отключить |
