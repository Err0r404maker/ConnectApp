## 🔧 Исправление проблемы с файлами

### Проблема
Файлы не отображаются в медиа галерее

### Причина
```typescript
// Фильтр проверяет fileUrl, но он может быть null
allFiles = messages.filter((msg: any) => {
  const hasFile = (msg.type === 'FILE' || msg.type === 'VOICE') && msg.fileUrl;
  return hasFile;
});
```

### Решение
Нужно также проверять `content` для base64 файлов:
```typescript
allFiles = messages.filter((msg: any) => {
  return (msg.type === 'FILE' || msg.type === 'VOICE') && 
         (msg.fileUrl || msg.content);
});
```
