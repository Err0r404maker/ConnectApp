## 😊 Реакции на сообщения

### Как это выглядит:

```
┌─────────────────────────────────────┐
│ Сообщение от пользователя           │
│ Текст сообщения здесь...            │
│                                     │
│ [👍 3] [❤️ 5] [😂 1]               │  ← Реакции
│                                     │
│ 10:30  ✓✓                          │
└─────────────────────────────────────┘
```

### При наведении:

```
┌─────────────────────────────────────────────────────┐
│ Сообщение от пользователя                           │
│ Текст сообщения здесь...                            │
│                                                     │
│ [👍] [❤️] [😂] [😮] [😢] [↩️] [📋] [→] [📌] [ℹ️]  │  ← Действия
│                                                     │
│ [👍 3] [❤️ 5] [😂 1]                               │
│ 10:30  ✓✓                                          │
└─────────────────────────────────────────────────────┘
```

### Использование:
1. Наведите курсор на сообщение
2. Появятся быстрые реакции
3. Кликните на эмодзи
4. Реакция добавится/удалится

## 📤 Пересылка сообщений

### Модальное окно:

```
┌─────────────────────────────────────────┐
│  Переслать сообщение                    │
│                                         │
│  ☑ Общий чат                           │
│  ☐ Группа 1А                           │
│  ☑ Преподаватели                       │
│  ☐ Друзья                              │
│                                         │
│  [Переслать (2)]  [Отмена]             │
└─────────────────────────────────────────┘
```

### Процесс:

```
1. Наведите на сообщение → нажмите →
2. Выберите чаты (галочки)
3. Нажмите "Переслать"
4. Сообщение появится в выбранных чатах
```

## 📝 Markdown форматирование

### Ввод:

```
**Жирный текст**
*Курсив*
`код`
[Ссылка](https://example.com)
```

### Отображение:

```
┌─────────────────────────────────────┐
│ Жирный текст                        │  ← Жирный
│ Курсив                              │  ← Курсив
│ код                                 │  ← Моноширинный
│ Ссылка                              │  ← Кликабельная
│                                     │
│ 10:30  ✓✓                          │
└─────────────────────────────────────┘
```

### Примеры использования:

```
Обычный текст: Привет!
Жирный: **Важно!**
Курсив: *Примечание*
Код: `console.log('Hello')`
Ссылка: [Google](https://google.com)

Комбинация:
**Важно:** *прочитайте* `README.md`
```

## 🎯 Горячие зоны интерфейса

```
┌─────────────────────────────────────────────────────────────┐
│  [1] Название    [2] [3] [4] [5] Поиск                      │  ← Header
├─────────────────────────────────────────────────────────────┤
│  [6] Закрепленные сообщения                                 │  ← Pins
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [7] Область сообщений                                      │  ← Messages
│      (скролл, виртуализация)                                │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  [8] Ответ на сообщение (если есть)                         │  ← Reply
├─────────────────────────────────────────────────────────────┤
│  [9] [10] [11] [12] Текст...  [13] Отправить                │  ← Input
└─────────────────────────────────────────────────────────────┘

Легенда:
[1] - Название чата
[2] - Участники (👥)
[3] - Настройки (⚙️)
[4] - Медиа (🖼️)
[5] - Поиск (🔍)
[6] - Закрепленные (📌)
[7] - Сообщения
[8] - Ответ (↩️)
[9] - Файлы (📎)
[10] - Эмодзи (😊)
[11] - Голос (🎤)
[12] - Текст
[13] - Отправить
```
